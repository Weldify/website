Hi, I'm Ian.

I've been making games since 2019. I started with Roblox, but later branched off into trying other game engines and even making fully standalone games.

RUNIC

Runic is a game jam game that I made in 3 days. It's written from scratch in the jai programming language, which is currently in closed beta. It uses a thin wrapper over OpenGL for rendering and a simple handwritten sound system I built using sokol_audio.

REVOLVER ROULETTE

Revolver Roulette is a game I made for the Facepunch game jam using the S&box engine. The attached video is timestamped and showcases a full server of players messing around!

I can't take all the credit for this one though! The revolver model was made by rekutami and the viewmodel animations were made by milspecy.

DRAWN TO LORDSKEEP

Drawn to Lordskeep is a prototype game I built over the course of a month in early 2024. Made using the Roblox game engine.

DILLARDS DESTINY

Dillards Destiny is a game I made for the Wowie 4.0 game jam. It's made using the Godot game engine.
